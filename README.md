# Bash-Orchestrated-Germline-Variant-Calling-Annotation-GATK4-ANNOVAR-
Bash-orchestrated GATK4 workflow for germline SNP/indel discovery (hg38), including alignment, BQSR, hard filtering, VCF normalization, and ANNOVAR annotation. Designed for reproducible variant analysis supporting inherited disease genetics and ACMG-oriented clinical interpretation.

# Bash-Orchestrated-Germline-Variant-Calling-Annotation-GATK4-ANNOVAR-
Bash-orchestrated GATK4 workflow for germline SNP/indel discovery (hg38), including alignment, BQSR, hard filtering, VCF normalization, and ANNOVAR annotation. Designed for reproducible variant analysis supporting inherited disease genetics, pharmacogenomics and ACMG-oriented clinical interpretation.

#Clinical Context — Germline Variant Calling (HaplotypeCaller)
Identifies inherited DNA variants present in all cells to diagnose genetic disease, assess hereditary risk, and guide medical management.
Example Applications:
Hereditary Cancer: BRCA1/2, TP53, RUNX1, GATA2
Rare Genetic Disease: SCN1A (epileptic encephalopathy), RAG1 (SCID immunodeficiency)
Carrier Screening: HBB, HBA1/HBA2 (thalassemia)
